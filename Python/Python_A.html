Pythonの概要

Pythonは、プログラムの読みやすさ・分かりやすさを重視したプログラミング言語です。これにより、初学者にとっての学習ハードルが他のプログラミング言語に比べて低く、挫折しにくい言語だと言えます。また、複数のプログラミングパラダイムに対応しているマルチパラダイムプログラミング言語でもあるため、オブジェクト指向を用いた堅牢なソフトウェア設計にも耐え得る仕様となっています。更に、インターネット上のライブラリを別途インストールすることで、Pythonの機能を拡張することができます。これにより、近年では、機械学習やWeb開発などにも利用されています。

読みやすさ・分かりやすさの秘密

Pythonには、オフサイドルールという規則があります。オフサイドルールは、字下げ（インデント）により文の固まりの範囲（ブロック）を示す規則です。分かりやすく言うと、私たちが日本語で文章を書く際、文章の初めに字下げをして、段落という固まりを作りますよね。これに似たものだと思っていただければ大丈夫です。
